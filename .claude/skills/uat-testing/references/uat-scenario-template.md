# UAT 用户场景模板

用户场景清单是 uat-testing 的核心工作产物，在 Step 1 读取 PRD 后构建，贯穿整个测试过程。

## 场景清单格式

```markdown
# UAT 用户场景清单 - {项目名称}

生成时间：{YYYY-MM-DD}
基准文档：PRD/{file}.md

---

## 用户角色定义

| 角色 | 描述 | 权限级别 | 测试账号 |
|------|------|---------|---------|
| 买家（Buyer） | 在平台购买商品的普通用户 | 普通用户 | buyer@test.com / Test1234! |
| 商家（Seller） | 在平台发布商品的卖家 | 商家权限 | seller@test.com / Test1234! |
| 管理员（Admin） | 后台管理人员 | 超级管理员 | admin@test.com / Admin1234! |
```

## 优先级定义

| 优先级 | 定义 | 处理 |
|--------|------|------|
| P0（核心路径） | 用户注册、登录、核心业务操作（购买/支付/核心功能） | 必须全部通过 |
| P1（重要功能） | 订单管理、个人中心、主要设置 | 强烈建议通过 |
| P2（辅助功能） | 边界情况、提示信息、非核心页面 | 记录，可延后修复 |

---

## 从 PRD 提取场景的方法

### 识别用户角色

从 PRD "用户角色"或"目标用户"章节提取，为每个角色创建：
- 描述（是谁？目标是什么？）
- 测试账号（需要在测试环境预创建）
- 核心操作路径（这个角色最重要的 2-3 个任务）

### 识别用户故事

从 PRD 中提取符合以下格式的内容：
- 明确的操作动词句（"用户可以..."、"买家能够..."）
- 验收标准（"当...时，系统显示..."）
- 业务流程图中的每个用户操作节点

**忽略**：纯技术实现描述（"系统将调用 API..."），这不是用户视角。

### 确定测试步骤

将每个用户故事分解为：
1. 用户去哪个页面（URL 或入口）
2. 用户做了什么操作（点击、填写、选择）
3. 用户看到什么结果（可见文字、页面跳转、提示消息）

**黄金原则**：步骤描述不能包含"调用 API"、"查数据库"——用户不知道这些。

## Playwright 测试代码示例

```typescript
import { test, expect } from '@playwright/test'

// U01: 买家注册账号
test('买家可以完成注册', async ({ page }) => {
  await page.goto('http://localhost:3000/register')

  // 填写注册表单
  await page.fill('[name="email"]', 'newbuyer@test.com')
  await page.fill('[name="password"]', 'Test1234!')
  await page.fill('[name="confirmPassword"]', 'Test1234!')
  await page.click('button[type="submit"]')

  // 验证用户视角结果
  await expect(page).toHaveURL(/\/welcome|\/dashboard/)
  await expect(page.locator('text=注册成功')).toBeVisible()
})

// U03 + U04: 买家完整购买流程
test('买家可以完成购买流程', async ({ page }) => {
  // 登录
  await page.goto('http://localhost:3000/login')
  await page.fill('[name="email"]', 'buyer@test.com')
  await page.fill('[name="password"]', 'Test1234!')
  await page.click('button[type="submit"]')

  // 搜索并添加到购物车
  await page.fill('[placeholder="搜索商品"]', '测试商品')
  await page.press('[placeholder="搜索商品"]', 'Enter')
  await page.click('.product-card >> nth=0')
  await page.click('text=加入购物车')
  await expect(page.locator('.cart-count')).toHaveText('1')

  // 结算
  await page.click('.cart-icon')
  await page.click('text=去结算')
  await page.fill('[name="address"]', '测试地址 123 号')
  await page.click('text=提交订单')

  // 验证结果
  await expect(page.locator('text=订单已提交')).toBeVisible()
  await expect(page).toHaveURL(/\/orders\/\d+/)
})
```
