## 2. 核心术语库（基于偏振片检测系统）

### 2.1 产品相关术语

#### 偏振片 (Polarizer)
- **定义**: 一种光学薄膜，能够将自然光转换为线性偏振光，是LCD显示屏的核心组件
- **英文**: Polarizer / Polarizing Film
- **应用场景**: LCD显示器、手机屏幕、笔记本电脑屏幕、车载显示
- **相关参数**: 偏振效率、透过率、消光比、耐久性
- **客户关注点**: 显示效果（对比度、亮度）、产品良率、生产成本
- **话术示例**: "偏振片的质量直接决定了显示屏的对比度和显示效果，我们的检测系统能够精确测量每片偏振片的关键指标，确保100%合格品出货。"

#### 消光比 (Extinction Ratio / ER)
- **定义**: 偏振片平行和垂直放置时透过光强度的比值，衡量偏振效果的核心指标
- **英文**: Extinction Ratio (ER), Contrast Ratio
- **公式**: ER = I_max / I_min (平行态光强/正交态光强)
- **典型值**: 优秀产品 ER > 50, 合格产品 ER > 40
- **应用场景**: 质量检验、工艺优化、产品分级
- **客户关注点**: 直接影响显示屏对比度，消光比越高显示效果越好
- **话术示例**: "消光比是评价偏振片质量的金标准。我们的系统可以测量4个波长的消光比，确保您的产品在不同光源下都有稳定表现。"

#### 透过率 (Transmittance)
- **定义**: 光线通过偏振片后的强度与入射光强度的比值，反映偏振片的光学效率
- **英文**: Transmittance / Transmission Rate
- **典型值**: 40-45%（单片偏振片）
- **影响因素**: 偏振膜材料、厚度、制造工艺
- **客户关注点**: 影响显示屏亮度和功耗
- **话术示例**: "透过率直接影响屏幕亮度。我们的系统通过测量初始光强和透过光强，可以精确计算透过率，帮助您优化工艺配方。"

#### 光强 (Optical Power / Light Intensity)
- **定义**: 单位时间内通过单位面积的光能量，用于定量描述光的强弱
- **英文**: Optical Power, Light Intensity, Radiant Flux
- **单位**: 瓦特(W)、毫瓦(mW)、纳瓦(nW)
- **测量工具**: 光功率计 (Optical Power Meter)
- **应用**: 消光比计算、透过率测量、光源稳定性监测
- **话术示例**: "我们集成了Thorlabs高精度光功率计，测量精度达到纳瓦级，确保数据可靠性。"

### 2.2 技术参数与规格

#### 多波长检测 (Multi-Wavelength Detection)
- **定义**: 使用多个激光波长（如1260nm, 1310nm, 1550nm, 1640nm）进行检测
- **英文**: Multi-Wavelength Testing
- **必要性**: 不同应用场景对特定波长的性能要求不同
- **我方产品配置**: 4个波长激光器，可独立控制
- **客户价值**: 一次性完成全光谱性能评估，无需更换设备
- **话术示例**: "我们配备4个波长的激光光源，覆盖近红外到红外波段，满足不同应用标准的检测需求。"

#### 9点检测模式 (9-Point Detection Mode)
- **定义**: 在每片偏振片上选取9个点位进行测量，获得面内均匀性数据
- **英文**: 9-Point Measurement Pattern
- **优势**: 更全面评估产品质量，发现局部缺陷
- **可配置性**: 可选1点、5点、9点模式
- **检测效率**: 9点模式耗时较长但准确率最高
- **客户关注点**: 大面积偏振片的均匀性保证
- **话术示例**: "相比单点检测，9点模式能够发现局部质量问题，避免不合格品流入下游，保护您的品牌声誉。"

#### 自动化运动控制 (Automated Motion Control)
- **定义**: 通过4轴运动平台（X/Y/Z/A）实现自动化样品定位和测量
- **英文**: Automated Motion Control System
- **轴数**: 4轴（XYZ三维移动 + A旋转）
- **控制精度**: 毫米级定位
- **批量检测**: 支持1-25片连续检测
- **客户价值**: 无需人工干预，提升检测效率和一致性
- **话术示例**: "我们的4轴运动平台可以自动完成25片偏振片的全自动检测，8小时无人值守运行，解放人力。"

#### 检测距离 (Measurement Distance)
- **定义**: 光源到偏振片、偏振片到探测器的距离
- **典型配置**: 5mm (近场), 35mm (远场)
- **影响**: 影响光束发散、测量精度
- **应用场景**: 近场适合薄样品，远场适合厚样品或特殊测试
- **话术示例**: "系统支持5mm和35mm两种检测距离，适应不同厚度样品的测试需求。"

### 2.3 系统架构术语

#### WPF (Windows Presentation Foundation)
- **定义**: 微软开发的图形界面框架，用于构建现代化用户界面
- **技术特点**: 数据绑定、矢量图形、动画支持
- **我方应用**: 软件界面开发
- **客户价值**: 现代化、直观的操作界面，降低培训成本

#### .NET Framework
- **版本**: 4.5-4.8
- **兼容性**: 支持Windows 7/8/10/11
- **稳定性**: 微软官方支持，长期维护

#### MySQL数据库
- **功能**: 存储检测数据、生成报表、历史追溯
- **客户价值**: 数据可追溯，支持质量分析和SPC统计

#### TCP通信
- **应用**: 软件模块间通信、远程监控
- **网络要求**: 需开放防火墙端口

### 2.4 运维相关术语

#### 回零 (Homing)
- **定义**: 运动平台返回机械原点，重新建立坐标系
- **使用场景**: 断电重启后、撞限位后、长期不用后
- **注意事项**: A轴（旋转轴）回零时需监控防止线缆扭曲
- **客户培训要点**: 重点讲解A轴回零的特殊操作

## 5. 系统操作要点（培训重点）

### 5.1 日常操作流程

1. **开机准备**
   - 检查电源、气源（如有）
   - 启动软件（自动连接运动控制器和激光器）
   - 确认"平台控制模块"显示"已连接"

2. **样品装载**
   - 打开防护罩
   - 按顺序放置偏振片（1-25号位）
   - 确保样品平整、无异物

3. **参数设置**
   - 选择检测波长（默认4个全选）
   - 选择检测距离（默认35mm）
   - 选择检测点数（默认9点）
   - 设置批次号和样品编号

4. **开始检测**
   - 点击"开始检测"按钮
   - 系统自动运行，实时显示进度
   - 检测完成后自动保存数据

5. **数据导出**
   - 点击"导出"按钮
   - 选择CSV格式
   - 保存到指定路径

### 5.2 故障排查速查表

| 故障现象 | 可能原因 | 解决方法 |
|---------|---------|---------|
| 软件启动失败 | 端口占用 | 重启电脑或关闭其他串口软件 |
| 运动平台不动 | 掉电/限位触发 | 检查电源，点击"回零"按钮 |
| 激光器未连接 | 通信端口错误 | 检查设置中的COM口配置 |
| 测量值异常（全0或超大） | 探头未连接/环境光干扰 | 检查探头连接，关闭环境光 |
| A轴回零线缆扭曲 | 回零方向错误 | 断电手动转回，重新上电 |
