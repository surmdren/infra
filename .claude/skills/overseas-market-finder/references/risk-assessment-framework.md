# 风险评估框架

海外市场进入的四维度风险评估体系，帮助识别和量化潜在风险。

## 风险评分体系

**评分范围**：0-100分（分数越高 = 风险越低 = 越安全）

**权重分配**：
- 政策风险：25%
- 运营风险：25%  
- 合规风险：25%
- 市场风险：25%

**综合风险等级**：
- 80-100分：低风险 ✅ - 可以进入
- 60-79分：中等风险 ⚠️ - 需要缓解措施
- 40-59分：较高风险 ⚠️⚠️ - 需要详细方案
- 0-39分：高风险 ❌ - 不建议进入

## 1. 政策风险 (25%)

### 1.1 关税风险
**评估维度**：
- 当前关税税率
- 历史税率变化趋势
- 贸易战影响
- 反倾销调查风险

**评分标准**：
```
关税率 ≤5%：+40分
关税率 5-15%：+25分
关税率 15-25%：+15分
关税率 >25%：+5分

贸易争端高风险品类（中美）：-20分
有反倾销历史：-15分
```

### 1.2 进口政策
**评估维度**：
- 进口配额限制
- 进口许可证要求
- 汇率管制政策
- 本地化政策压力

**评分标准**：
```
无限制：+30分
轻度限制：+20分
中度限制：+10分
严格限制：+0分
```

### 1.3 双边关系
**评估维度**：
- 中国与目标国政治关系
- 贸易协定状况
- 外交风险事件
- 媒体舆论环境

**评分标准**：
```
关系良好：+30分
关系一般：+15分
关系紧张：+5分
关系恶化：+0分
```

## 2. 运营风险 (25%)

### 2.1 汇率风险
**评估维度**：
- 目标国货币稳定性
- 汇率历史波动率
- 央行政策倾向
- 通胀水平

**评分标准**：
```
低波动率(<5%年化)：+30分
中等波动率(5-15%)：+20分
高波动率(>15%)：+10分
货币危机风险：+0分
```

### 2.2 物流风险
**评估维度**：
- 物流成本占比
- 运输时效稳定性
- 海关清关效率
- 基础设施完善度

**评分标准**：
```
物流成本<5%产品价值：+25分
物流成本5-10%：+15分
物流成本10-20%：+10分
物流成本>20%：+0分

发达国家基础设施：+额外10分
```

### 2.3 支付风险
**评估维度**：
- 客户信用状况
- 账期长短
- 收款保障机制
- 银行系统稳定性

**评分标准**：
```
即期付款(T/T)：+25分
30天账期：+20分
60-90天账期：+10分
120天+账期：+0分

有信用保险：+额外10分
```

### 2.4 供应链风险
**评估维度**：
- 供应商集中度
- 原料价格波动
- 生产能力稳定性
- 替代方案可行性

**评分标准**：
```
多供应商分散：+20分
2-3个供应商：+15分
单一供应商：+5分

有备用产地：+额外5分
```

## 3. 合规风险 (25%)

### 3.1 知识产权风险
**评估维度**：
- 专利侵权风险
- 商标冲突可能
- 版权争议
- 技术壁垒

**评分标准**：
```
FTO分析清洁：+30分
小概率风险：+20分
中等风险：+10分
高侵权风险：+0分

有专利授权：+额外10分
```

### 3.2 产品认证
**评估维度**：
- 强制性认证要求
- 认证时间和成本
- 技术门槛高低
- 本地测试要求

**评分标准**：
```
无强制认证：+25分
1-2项认证：+15分
3-5项认证：+10分
复杂认证体系：+0分

已有认证：+额外15分
```

### 3.3 行业监管
**评估维度**：
- 行业监管严格度
- 准入门槛高低
- 合规成本
- 政策变化频率

**评分标准**：
```
轻监管行业：+25分
一般监管：+15分
严格监管：+5分
极严监管(医疗/食品)：+0分
```

### 3.4 数据合规
**评估维度**：
- 数据保护法规
- 本地化存储要求
- 跨境数据传输
- 用户隐私保护

**评分标准**：
```
无特殊要求：+20分
一般要求：+15分
GDPR级别要求：+5分
极严格要求：+0分
```

## 4. 市场风险 (25%)

### 4.1 文化适配
**评估维度**：
- 产品本地化需求
- 文化差异程度
- 消费习惯差异
- 宗教禁忌影响

**评分标准**：
```
无需本地化：+30分
轻度本地化：+20分
重度本地化：+10分
完全重新设计：+0分
```

### 4.2 竞争强度
**评估维度**：
- 竞争对手数量
- 市场份额分布
- 品牌壁垒高度
- 价格战可能性

**评分标准**：
```
竞争对手<10个：+25分
竞争对手10-30个：+15分
竞争对手30-50个：+8分
竞争对手>50个：+0分

无强势品牌：+额外10分
```

### 4.3 市场成熟度
**评估维度**：
- 产品生命周期阶段
- 市场教育程度
- 消费者认知度
- 渠道建设完善度

**评分标准**：
```
成熟市场：+20分
成长期市场：+25分
导入期市场：+10分
衰退期市场：+0分
```

### 4.4 渠道风险
**评估维度**：
- 渠道伙伴稳定性
- 渠道成本和效率
- 多渠道依赖度
- 去中介化威胁

**评分标准**：
```
多渠道分散：+25分
2-3个主渠道：+15分
单一渠道依赖：+5分

直营渠道控制：+额外10分
```

## 风险缓解策略

### 政策风险缓解
```
1. 多元化市场布局（不依赖单一市场）
2. 供应链多元化（东南亚代工）
3. 本地化合作（当地合资伙伴）
4. 政策保险（出口信用险）
```

### 运营风险缓解
```
1. 汇率对冲（远期合约、币种匹配）
2. 供应链备份（多供应商、安全库存）
3. 收款保障（信用保险、保理）
4. 成本控制（精益管理、规模效应）
```

### 合规风险缓解
```
1. 合规前置（提前法务审查）
2. 专业咨询（当地律所、会计师）
3. 认证预投（提前获得关键认证）
4. 合规投资（建立合规体系）
```

### 市场风险缓解
```
1. 市场细分（找准目标用户）
2. 差异化定位（避免价格战）
3. 本地化投资（适配当地需求）
4. 品牌建设（建立竞争壁垒）
```

## 风险监控指标

### 实时监控
```
1. 汇率每日监控（设置预警线）
2. 政策新闻跟踪（贸易政策变化）
3. 竞争对手动态（价格、产品变化）
4. 客户信用监控（逾期率变化）
```

### 定期评估
```
1. 月度运营指标（销量、毛利、CAC）
2. 季度风险复评（政策、竞争环境）
3. 年度战略调整（市场地位、投入产出）
```

## 决策矩阵

### 风险-收益平衡
```
高收益+低风险：立即进入 ✅
高收益+中风险：制定缓解方案后进入 ⚠️
高收益+高风险：小规模测试 🔍
低收益+高风险：不进入 ❌
```

### 风险承受能力匹配
```
保守型企业：仅进入低风险市场
平衡型企业：可承受中等风险
激进型企业：可承受高风险高回报
```
