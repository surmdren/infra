# 财务模型说明

海外市场机会财务评估框架，用于量化项目投资回报和风险。

## 核心指标体系

### 1. ROI (投资回报率)
```
ROI = (净收益 - 总投资) / 总投资 × 100%

计算周期：
- 12个月 ROI：短期回报
- 24个月 ROI：中期回报  
- 36个月 ROI：长期回报
```

### 2. NPV (净现值)
```
NPV = Σ(现金流 / (1+折现率)^期间) - 初始投资

折现率设定：
- 8%：低风险市场（欧美成熟市场）
- 12%：中等风险市场（东南亚、拉美）
- 15%：高风险市场（新兴市场）
```

### 3. 回本周期 (Payback Period)
```
累计现金流转正的时间点

分类标准：
- ≤12个月：优秀
- 13-18个月：良好
- 19-24个月：可接受
- >24个月：谨慎
```

### 4. IRR (内部收益率)
```
NPV = 0 时的折现率

判断标准：
- >25%：高收益项目
- 15%-25%：中等收益项目
- 8%-15%：低收益项目
- <8%：不建议投资
```

## 三种场景分析

### 保守场景 (Conservative)
**假设条件**：
- 市场渗透率：目标的60%
- 平均客单价：预期的80%
- 毛利率：预期的90%
- 市场增长：预期的50%
- 竞争加剧：预期的150%

**风险因子**：1.5x（安全边际）

### 正常场景 (Normal)
**假设条件**：
- 市场渗透率：目标的100%
- 平均客单价：预期的100%
- 毛利率：预期的100%
- 市场增长：预期的100%
- 竞争水平：预期的100%

**风险因子**：1.0x（基准情况）

### 乐观场景 (Optimistic)
**假设条件**：
- 市场渗透率：目标的130%
- 平均客单价：预期的120%
- 毛利率：预期的110%
- 市场增长：预期的150%
- 竞争程度：预期的80%

**风险因子**：0.8x（最优情况）

## 成本结构模型

### 一次性成本
```
1. 产品开发成本
   - R&D费用
   - 认证费用（FDA、CE、FCC等）
   - 知识产权成本

2. 市场准入成本
   - 法律设立成本
   - 首批库存
   - 渠道保证金

3. 营销启动成本
   - 品牌建设
   - 首轮广告投放
   - KOL合作费用
```

### 持续性成本
```
1. 采购成本
   - 原材料/产品采购
   - 运输物流费用
   - 关税税费

2. 运营成本
   - 人员工资
   - 办公租金
   - 系统维护费

3. 营销成本
   - 广告投放（月度）
   - 销售提成
   - 客服成本

4. 合规成本
   - 税务会计
   - 法律咨询
   - 保险费用
```

## 收入预测模型

### 销量预测
```
月销量 = 目标市场规模 × 市场渗透率 × 市场份额

考虑因素：
- 产品生命周期（导入期→成长期→成熟期→衰退期）
- 季节性波动
- 竞争对手反应
- 经济周期影响
```

### 定价策略
```
1. 渗透定价（低价抢市场）
   - 适用：价格敏感市场、规模效应明显
   - 风险：价格战、盈利压力

2. 撇脂定价（高价高毛利）
   - 适用：创新产品、品牌溢价
   - 风险：市场接受度、竞争模仿

3. 竞争定价（跟随市场）
   - 适用：标准化产品、成熟市场
   - 风险：价格透明、利润微薄
```

### 客户获取
```
CAC (客户获取成本) = 营销费用 / 新增客户数

LTV (客户生命周期价值) = 平均客单价 × 复购次数 × 毛利率

LTV/CAC 比例：
- >3:1：健康
- 2-3:1：可接受
- <2:1：不可持续
```

## 风险调整

### 汇率风险
```
影响：收入（外币）vs 成本（人民币）

对冲策略：
- 远期外汇合约
- 币种匹配（外币采购）
- 价格调整机制
```

### 政策风险
```
关税变化影响：
- 10%关税变化 ≈ 5-8%毛利率影响
- 考虑转嫁给客户的能力
- 备选供应地方案（东南亚）
```

### 竞争风险
```
市场份额下滑影响：
- 新进入者：-10-20%销量
- 价格战：-15-25%毛利
- 技术迭代：-30-50%销量
```

## 敏感性分析

### 关键变量影响
```
销量变化±10% → ROI变化±15-25%
客单价变化±10% → ROI变化±20-30%  
毛利率变化±5% → ROI变化±25-35%
CAC变化±20% → ROI变化±10-15%
```

### 盈亏平衡分析
```
盈亏平衡点 = 固定成本 / (单价 - 变动成本)

安全边际 = (预期销量 - 盈亏平衡销量) / 预期销量
```

## 决策框架

### 财务门槛
```
必须满足：
- 12个月ROI ≥ 30%（保守场景）
- NPV > 0（正常场景，12%折现率）
- 回本周期 ≤ 18个月

优秀项目：
- 12个月ROI ≥ 50%
- IRR ≥ 25%
- 回本周期 ≤ 12个月
```

### 风险承受度
```
低风险偏好：只考虑保守场景
中等风险：正常场景为主，保守场景兜底
高风险偏好：乐观场景为主，正常场景兜底
```
