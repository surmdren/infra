---

## 四段式脚本结构

| 段落          | 时长       | 内容                         |
| ----------- | -------- | -------------------------- |
| **Hook**    | 0–30s    | 结果先行，直接展示 AI 自动化最终产物       |
| **The Gap** | 30s–2min | 锚定痛点，描述手动操作有多慢多贵           |
| **Build**   | 核心部分     | 环境准备 → API配置 → 逻辑构建 → 运行测试 |
| **CTA**     | 结尾       | 引导下载 Prompt 模板或加入社群        |

## 主题色规范

| 用途   | 颜色          | Hex                   |
| ---- | ----------- | --------------------- |
| 背景   | 深灰/黑        | `#0f0f0f` 或 `#1a1a2e` |
| 强调色1 | 青绿色 Cyan    | `#00d4ff`             |
| 强调色2 | 紫红色 Magenta | `#ff00ff`             |
| 正文   | 白色          | `#ffffff`             |
| 成功状态 | 绿色          | `#00ff88`             |
| 错误状态 | 红色          | `#ff4444`             |

**架构图节点颜色约定：**
- Cyan = 输入
- Magenta = 输出
- 白色 = 处理/中间步骤

所有工具统一使用同一色板：Eraser、Mermaid、剪映字幕、Figma 封面。

> 一次定好，全平台视觉一致，不用每次重新想。

## 标志性视觉元素

- **霓虹发光效果**：文字和图标周围加 Cyan/Magenta 光晕
- **毛玻璃效果**：UI 窗口半透明背景 + 模糊
- **文字动效**：淡入 + 轻微位移，不过度

---

## 个人 IP 差异化（在 Nate 基础上优化）

- **封面必须有你的真人头像**：让观众记住人，不只是风格
- **封面加上频道名/你的名字**：统一品牌识别
- **中英双语字幕**：服务英文母语观众 + 学英文的中文观众，Nate 没有做
- **架构图全部用 Eraser.io 同一模板**：比 Nate 更统一

---

## 封面模板规范

- 尺寸：**1280 × 720px**
- 背景：深灰/黑（`#0f0f0f`）
- 左侧：真人照片（抠图去背景）
- 右侧：主标题文字（**Poppins Bold**，80–100px，白色 + Cyan 发光描边）
- 点缀：工具 Logo 或结果截图
- 必须包含：你的名字或频道名

---

## 画中画规范

- **形状**：圆形裁切
- **大小**：画面宽度的 20–25%
- **位置**：右下角（不遮挡代码/演示区域）
- **边框**：Cyan `#00d4ff`，2–3px
- **显示时机**：Hook 开场、讲解段落显示；实操演示时可隐藏
- **背景**：绿幕抠图或自然背景，保持简洁

---

## 片头/片尾规范

### 片头
- 时长：**3–5秒**
- 内容：频道 Logo 动画 + 频道名
- 风格：Cyan 发光效果，淡入
- ❌ 不要超过5秒，观众会跳过

### 片尾
- 时长：**15–20秒**
- 布局：左侧推荐视频 + 右侧订阅按钮
- CTA 文字：统一用白色 Poppins Bold
- 必须包含：社群/资源链接二维码（后期 Skool 建好后加入）

## 色彩搭配最佳实践

### 核心比例法则
- **背景**：占画面 70%（深灰/黑）
- **白色正文**：占 20%
- **Cyan + Magenta 点缀**：占 10%（不能用太多，否则失去冲击力）

### 封面配色
- 背景用 `#0f0f0f` 纯黑
- 主标题：白色 + Cyan 发光描边
- 真人照片：自然色，不加滤镜
- 工具 Logo/截图：保持原色，作为点缀
- ❌ 不要：背景用亮色、文字用 Magenta（太刺眼）

### 架构图配色
- 背景：`#1a1a2e` 深蓝黑
- 输入节点：Cyan `#00d4ff` 边框 + 发光
- 输出节点：Magenta `#ff00ff` 边框 + 发光
- 中间处理节点：白色边框
- 连接线：白色或浅灰
- 成功路径：绿色 `#00ff88`
- 错误路径：红色 `#ff4444`

### 字幕/标注配色
- 字幕文字：白色，黑色描边
- 关键词高亮：Cyan
- 警告/注意：Magenta
- 箭头/标注框：白色或 Cyan

### 避坑清单
- ❌ Cyan 和 Magenta 不要同时大面积出现，只做点缀
- ❌ 不要在深色背景上用深色文字
- ❌ 封面不要超过 3 种颜色（背景 + 白色 + 一个强调色）
- ❌ 代码块不要改变背景色，保持 IDE 默认深色即可
