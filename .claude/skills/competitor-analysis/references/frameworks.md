# 分析框架参考

竞争分析框架的详细指南。

## SWOT 分析

### 模板

```markdown
## SWOT 分析：[公司/产品名称]

### 优势（内部优势）
1. **[优势类别]** - [具体优势]
   - 影响：[对竞争的影响]
   - 可持续性：[是否可持续/易被复制]

### 劣势（内部劣势）
1. **[劣势类别]** - [具体劣势]
   - 影响：[对竞争的影响]
   - 紧急度：[修复优先级]

### 机会（外部机会）
1. **[机会类别]** - [具体机会]
   - 市场规模：[市场规模估算]
   - 时间窗口：[机会窗口期]

### 威胁（外部威胁）
1. **[威胁类别]** - [具体威胁]
   - 发生概率：[发生概率]
   - 潜在影响：[潜在影响]

### 战略启示
- [基于 SWOT 的战略建议]
```

### 使用要点
- 具体且数据驱动
- 按影响力和紧迫性排序
- 聚焦可执行的洞察
- 每季度更新

## 波特五力模型

### 模板

```markdown
## 波特五力分析：[行业名称]

### 1. 供应商议价能力
**强度：** 高/中/低
- **主要供应商：** [供应商列表]
- **转换成本：** [转换成本高低]
- **前向整合风险：** [供应商前向整合风险]
- **影响：** [对议价能力的影响]

### 2. 买家议价能力
**强度：** 高/中/低
- **买家集中度：** [买家集中度]
- **价格敏感度：** [价格敏感度]
- **转换成本：** [转换成本]
- **产品差异化：** [产品差异化程度]
- **影响：** [对定价策略的影响]

### 3. 同业竞争强度
**强度：** 高/中/低
- **竞争对手数量：** [竞争对手数量]
- **行业增长率：** [行业增长率]
- **退出壁垒：** [退出壁垒]
- **固定成本占比：** [固定成本占比]
- **影响：** [对竞争强度的影响]

### 4. 新进入者威胁
**强度：** 高/中/低
- **资本要求：** [资金需求]
- **规模经济：** [规模经济]
- **品牌忠诚度：** [品牌忠诚度]
- **渠道控制：** [渠道控制]
- **监管壁垒：** [监管壁垒]
- **影响：** [对新进入者的防御]

### 5. 替代品威胁
**强度：** 高/中/低
- **替代产品：** [替代品列表]
- **性价比：** [性价比对比]
- **转换成本：** [转换成本]
- **影响：** [被替代的风险]

### 总体吸引力
**行业吸引力：** [吸引力评级]
- **盈利潜力：** [盈利潜力]
- **增长前景：** [增长前景]
- **建议策略：** [基于五力的战略建议]
```

### 解读指南

| 力量得分 | 战略启示 |
|---------|---------|
| 多数力量不利 | 行业缺乏吸引力，避免进入 |
| 力量参半 | 中等吸引力，需要差异化 |
| 多数力量有利 | 行业有吸引力，适合进入 |

## 价值主张画布

### 模板

```markdown
## 价值主张画布：[产品名称]

### 客户概况
| 客户工作 | 痛点 | 收益 |
|----------|-----|------|
| [客户试图完成的任务] | [负面结果、风险、障碍] | [正面结果、收益、愿望] |
| | | |

### 价值图
| 产品与服务 | 痛点缓解 | 收益创造 |
|-----------|---------|---------|
| [我们提供的] | [如何缓解痛点] | [如何创造收益] |
| | | |

### 匹配度分析
**痛点匹配度：** [我们是否很好地缓解了主要客户痛点？]
- [痛点 1] → [我们的解决方案] ✓/✗
- [痛点 2] → [我们的解决方案] ✓/✗

**收益匹配度：** [我们是否创造了客户期望的收益？]
- [收益 1] → [我们的解决方案] ✓/✗
- [收益 2] → [我们的解决方案] ✓/✗

**整体匹配度评分：** [评分 1-10]

### 竞争对比
| 竞品 | 痛点匹配度 | 收益匹配度 | 整体 |
|------|-----------|-----------|------|
| 我们 | [X]/10 | [Y]/10 | [Z]/10 |
| 竞品 A | [X]/10 | [Y]/10 | [Z]/10 |
| 竞品 B | [X]/10 | [Y]/10 | [Z]/10 |
```

### 使用提示
1. 通过访谈真实客户填充
2. 具体明确，不笼统
3. 聚焦最重要的工作/痛点/收益
4. 基于客户反馈更新

## 蓝海战略 ERCC 矩阵

### 模板

```markdown
## 蓝海战略：[产品/市场]

### 当前行业逻辑
**行业竞争要素：**
- 要素 1
- 要素 2
- 要素 3
- ...

### ERCC 矩阵

| | 剔除 | 减少 | 提高 | 创造 |
|---|-----------|--------|-------|------|
| **要素** | [应剔除哪些要素？] | [应降至行业标准以下？] | [应提至行业标准以上？] | [应创造哪些从未存在的要素？] |

### 新价值曲线
在每个要素上按 10 分制绘制：

| 要素 | 行业 | 我们 |
|------|------|------|
| [要素 1] | 5 | 1 |
| [要素 2] | 8 | 3 |
| [要素 3] | 3 | 9 |
| [要素 4] | 6 | 10 |

### 战略逻辑
**叙述版：**
- [用 2-3 段解释战略逻辑]
- [为何能创造无人竞争的市场空间？]
- [我们做了哪些取舍？]

### 验证
- [进行的客户访谈]
- [对新价值主张的反馈]
- [做出的调整决策]
```

### 四步动作框架

**剔除** - 行业理所当然的要素应剔除哪些？
- 示例：太阳马戏团剔除了动物表演

**减少** - 哪些要素应降至行业标准以下？
- 示例：西南航空减少座位选择、餐饮、枢纽辐射

**提高** - 哪些要素应提至行业标准以上？
- 示例：星巴克提高咖啡品质、店堂环境

**创造** - 应创造哪些前所未有的要素？
- 示例：太阳马戏团创造了剧场体验

## 市场定位图

### 模板

```markdown
## 市场定位图：[行业]

### 维度（选择 2-3 个）
- X 轴：[价格 低 → 高]
- Y 轴：[品质 基础 → 高端]
- 气泡大小：[市场份额]

### 定位图

[竞争格局的可视化表示]

### 象限分析
| 象限 | 特征 | 玩有者 | 策略 |
|------|------|--------|------|
| 高价高质 | 高端市场 | [竞品] | 差异化 |
| 低价低价质 | 大众市场 | [竞品] | 成本领先 |
| 高价低质 | [利基/失败] | [竞品] | 避免 |
| 低价高质 | [最佳位置] | [竞品] | 最佳价值 |

### 我们的定位
- **当前位置：** [我们现在哪里]
- **理想位置：** [我们想去哪里]
- **路径：** [如何到达]
```
