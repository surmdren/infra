# 示例图片目录

本目录用于存放鞋子产品图的示例（优化前/优化后对比）。

## 目录结构

```
examples/
├── before/                 # 优化前的原图
│   ├── sneakers/           # 运动鞋示例
│   ├── leather_shoes/      # 皮鞋示例
│   ├── high_heels/         # 高跟鞋示例
│   ├── canvas_shoes/       # 帆布鞋示例
│   └── boots/              # 靴子示例
├── after/                  # 优化后的图片
│   ├── sneakers/
│   ├── leather_shoes/
│   ├── high_heels/
│   ├── canvas_shoes/
│   └── boots/
└── comparisons/            # 对比图（before vs after）
    └── side_by_side/
```

## 示例类型

### 1. 背景移除示例

**Before**:
- 复杂背景（室内、户外、桌面等）
- 有阴影、杂物、背景纹理

**After**:
- 纯白底 RGB(255,255,255)
- 鞋子边缘清晰
- 无背景残留

**文件命名**: `bg_removal_sneakers_01_before.jpg` / `bg_removal_sneakers_01_after.jpg`

---

### 2. 材质增强示例

**Before**:
- 材质纹理不清晰（皮革纹路模糊、网布透气孔不明显）
- 光泽不自然（塑料感、过亮或过暗）

**After**:
- 材质纹理清晰可见
- 光泽自然真实
- 细节锐利

**文件命名**: `material_enhance_leather_01_before.jpg` / `material_enhance_leather_01_after.jpg`

---

### 3. 光线优化示例

**Before**:
- 光线不均匀（局部过曝或过暗）
- 阴影过硬或过重
- 整体偏暗或偏亮

**After**:
- 光线明亮均匀
- 阴影柔和自然
- 曝光适中

**文件命名**: `lighting_fix_heels_01_before.jpg` / `lighting_fix_heels_01_after.jpg`

---

### 4. 全面优化示例

**Before**:
- 手机拍摄的鞋子照片（背景杂乱、光线不佳、角度随意）
- 工厂样图（质量一般、背景简陋）

**After**:
- 纯白底电商主图
- 专业光线
- 材质清晰
- 符合平台标准

**文件命名**: `full_enhance_sneakers_01_before.jpg` / `full_enhance_sneakers_01_after.jpg`

---

## 常见问题修复示例

### 白色鞋子与白底融合

**Before**: 白色鞋子与白底边界不清
**After**: 增加淡灰色阴影，鞋子边缘清晰可见

### 高跟鞋鞋跟变形

**Before**: 鞋跟弯曲或变形
**After**: 鞋跟笔直，形状正确

### 多材质混乱

**Before**: 网布、皮革、橡胶材质融合不清
**After**: 每种材质清晰区分，纹理各异

---

## 示例图片要求

1. **分辨率**: 原图至少 800×800，优化后建议 1200×1200
2. **格式**: JPG（品质 90%）或 PNG
3. **文件大小**: 单张图片 < 2MB
4. **多样性**: 涵盖不同鞋款、不同材质、不同问题类型
5. **真实性**: 使用真实的优化前/优化后图片，不使用 mock 图
